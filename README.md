## 🧠 Overview

**SnipDo** is a lightweight productivity application that adds quick actions to selected text anywhere in Windows.

It helps users perform searches, translations, formatting, clipboard operations, and other tasks from a convenient popup menu.

The software supports customizable extensions, making it easy to tailor available actions for different workflows.

Perfect for professionals, students, developers, and anyone who regularly works with text.
